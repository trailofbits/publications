title: "Mutation Testing with Slither: A New Way to Find High-Severity Issues"
date: 2025-11
authors:
  - Guillermo Larregay
conference:
  - DeFi Security Summit 2025
---

This presentation introduces mutation testing capabilities in Slither, demonstrating a new approach to finding high-severity vulnerabilities in smart contracts. The talk shows examples of real-life usage of slither-mutate, illustrating how mutation testing can reveal blind spots in test suites that traditional coverage metrics miss.
