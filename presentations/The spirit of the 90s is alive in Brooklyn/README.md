Using the tools and techniques of today to solve problems that only existed in the 90s and are still alive in DEFCON CTF. This talk explains and demos tools built on Binary Ninja's BNIL to find 90s-era bugs such as format string vulnerabilities, stack buffer overflows, and command injection.
