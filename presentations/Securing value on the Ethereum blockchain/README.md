Dan Guido discusses Trail of Bits' industry-leading blockchain security services for smart contracts, asset custody, and blockchain design. Using deep expertise in cryptography, compilers, and program analysis, Trail of Bits has sought to secure the foundation of blockchain infrastructure by building better tools and frameworks, particularly for Ethereum smart contracts.
