Buttercup is an automated, AI-powered system developed by Trail of Bits that finds and fixes bugs in open-source software. It won 2nd place and a $3 million prize in DARPA's AI Cyber Challenge. This talk explains how Buttercup found and patched diverse vulnerabilities with high accuracy and demonstrates how to use the open-source Buttercup system.
