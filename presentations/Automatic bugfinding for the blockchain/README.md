This presentation covers the technical aspects of the Ethereum Virtual Machine (EVM) and provides an overview of the most common smart contract bugs. It then discusses the design and implementation of Manticore, an EVM-capable dynamic symbolic execution engine that enables human-assisted analysis and automatic detection of vulnerabilities.
