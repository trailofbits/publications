Your exchange or protocol's cold wallet has been hacked. For most orgs, this would be a game-over scenario. This talk discusses how to build safer, more robust cold storage solutions on Ethereum that can even endure multisig compromise, using smart contract programmability to enforce security policies at the protocol level.
